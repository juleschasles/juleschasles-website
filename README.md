# juleschasles.com

Personal website for Jules Chasles - minimalist, readability-optimized design.

## Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Content**: MDX for blog posts
- **Deployment**: Vercel
- **Domain**: juleschasles.com

## Design Principles

1. **Readability First**: Georgia serif, 20px body text, 1.8 line height
2. **Speed**: Static generation, < 1s load time
3. **Simplicity**: Content-first, minimal dependencies
4. **Easy Maintenance**: Update content without touching code

## Color Scheme

- Background: `#F8F8F6` (cool off-white)
- Text: `#2A2A2A` (soft black)
- Secondary: `#5A5A5A`
- Max line length: 65 characters

## Local Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Push to `main` branch → auto-deploys to Vercel

## Content Management

Add blog posts as `.mdx` files in `/content/posts/`:

```markdown
---
title: "Post Title"
date: "2025-01-15"
slug: "post-title"
---

Your content here...
```
